## O que é o projeto
O projeto é um aplicativo para exibir a localização do usuário e permitir que ele a salve

## Bibliotecas usadas
As bibliotecas usadas no projeto foram:
- Expo Location
- React Native Maps
- EAS (Expo Application Service) - build do projeto
